### Dev
```
$ npm install
$ node index.js
```

### Commands
- `!album keyword1 (keywords2) ...`
- `!clear`
- `!help`
- `!jump pos`
- `!lyric`
- `!next`
- `!pause`
- `!ping`
- `!play keyword1 (keywords2) ...`
- `!play youtube_url`
- `!playfile uploaded_file`
- `!playlist`
- `!prev`
- `!queue (amount)`
- `!remove pos (pos2) (pos3->pos4) ...`
- `!resume`
- `!shuffle`
- `!stop`
- `!user username`

use `!help <command>` to check detailed explaination
