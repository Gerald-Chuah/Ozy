# Ozy

从网易云播放的discord机器人

[Changelog | 更新日志](https://github.com/k27dong/Ozy/blob/main/CHANGELOG.md)

### Dev
```
$ npm install
$ node index.js
```

### Commands
- `!album keyword1 (keywords2) ...`
- `!clear`
- `!help`
- `!jump pos`
- `!lyric`
- `!next`
- `!pause`
- `!ping`
- `!play keyword1 (keywords2) ...`
- `!play youtube_url`
- `!playfile uploaded_file`
- `!playlist`
- `!prev`
- `!queue (amount)`
- `!remove pos (pos2) (pos3->pos4) ...`
- `!resume`
- `!shuffle`
- `!stop`
- `!user username`
